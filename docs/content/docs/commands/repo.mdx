## ワークフロー

リポジトリファイルを使った基本的なワークフロー：

### 1. リポジトリファイルを作成

<Callout type="info">
`mpm create-repo`コマンドは現在実装予定です。現時点では手動でJSONファイルを作成してください。
</Callout>

手動でJSONファイルを作成：

```json
{
    "id": "QuickShop",
    "website": "https://modrinth.com/plugin/quickshop-hikari",
    "repositories": [
        {
            "type": "modrinth",
            "id": "quickshop-hikari"
        }
    ]
}
```

### 2. プラグインを追加

```bash
/mpm add QuickShop
```

### 3. インストール

```bash
/mpm install
```

## トラブルシューティング

### ファイル選択パターン（fileNameRegex）が機能しない

1. **正規表現の構文を確認**
   ```json
   // 正しい例
   "fileNameRegex": "EssentialsX-.*\\.jar"
   "fileNameRegex": "QuickShop-Hikari-[0-9\\.]+\\.jar"

   // エスケープを忘れないように（ドットは \\.）
   ```

2. **パターンをシンプルにしてテスト**
   ```json
   // まずシンプルなパターンから
   "fileNameRegex": ".*\\.jar"
   ```

### リポジトリファイルが見つからない

```bash
# ディレクトリ構造を確認
ls plugins/MinecraftPluginManager/repository/

# ファイル名がプラグインIDと一致しているか確認
cat plugins/MinecraftPluginManager/repository/QuickShop.json
```
