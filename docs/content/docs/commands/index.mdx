## 📖 基本的な使い方

### ワークフロー

1. **リポジトリファイルを作成**（手動でJSONファイルを作成）

2. **プラグインを追加**
   ```bash
   /mpm add QuickShop
   ```

3. **プラグインをインストール**
   ```bash
   /mpm install
   ```

### プラグインの管理

- **一覧表示**: `/mpm list` でインストール済みプラグインを確認
- **更新確認**: `/mpm outdated <plugin>` で個別確認、`/mpm outdatedAll` で一括確認
- **更新実行**: `/mpm update` で更新を適用
- **アンインストール**: `/mpm remove <plugin>` でプラグインを削除

### 旧ワークフロー（非推奨）

URLから直接インストール:
```bash
/mpm install https://modrinth.com/plugin/quickshop-hikari
```

<Callout type="warn" title="重要な変更">
`mpm install`コマンドは引数なしで実行するように変更されました。
URL指定のインストールは削除されました。
新しいワークフロー（リポジトリファイル作成 → `mpm add` → `mpm install`）への移行をお願いします。
</Callout>

<Callout type="info" title="管理外プラグインについて">
mpm管理下にないプラグインは、`mpm removeUnmanaged`で削除されます。
管理外プラグインを保持したい場合は、`/plugins/minecraftPluginManager/local`に移動してください。
`mpm install`実行時に自動的にコピーされます。
</Callout>
