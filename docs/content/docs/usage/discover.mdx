mpm では、キーワードによるプラグインの検索と、プラグインの詳細情報の表示ができます。導入したいプラグインを探したり、バージョンやライセンスを確認したりするのに役立ちます。

## 🔍 プラグインを検索する

`mpm search <query>` で、複数のリポジトリ（Modrinth・Hangar・SpigotMC・GitHub）を横断してプラグインを検索します。結果はダウンロード数の多い順に並びます。

```bash
mpm search voice
mpm search shop --limit 20
```

- `--limit <数>` — 表示件数の上限（デフォルト: 10）

各結果には、リポジトリ種別・プラグイン名・ダウンロード数・識別子（id）・概要・プロジェクトページのURLが表示されます。

<Callout type="info" title="インストールについて">
mpm はリポジトリファイルに基づいてプラグインを管理します。検索で見つけたプラグインをインストールするには、そのプラグイン用のリポジトリファイルを用意してから `mpm add` を実行してください。検索結果に表示される URL は、リポジトリファイルを作成する際の参考になります。
</Callout>

## ℹ️ プラグインの詳細を表示する

`mpm info <plugin>` で、リポジトリファイルに定義されたプラグインの詳細情報を表示します。

```bash
mpm info QuickShop
```

表示される情報:

- **ソース** — リポジトリ種別と識別子
- **状態** — 未インストール / インストール済み（バージョン）/ ロック中
- **最新バージョン**
- **ダウンロード数**
- **ライセンス**
- **ホームページ**
- **説明**

取得できなかった項目は省略されます。
